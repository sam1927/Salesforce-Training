# Salesforce Training (MkDocs)

This project has been tidied for easier maintenance without changing the rendered content.

## Structure

- `docs/` – MkDocs pages
- `docs/assets/images/` – organised page and site images/GIFs
- `docs/assets/stylesheets/` – custom CSS
- `docs/assets/javascripts/` – custom JavaScript
- `housekeeping/notes/` – internal scratch files kept out of the published site

## Notes

- Asset paths were updated to match the new organised folders.
- Visible wording, layout, and visuals were intentionally left unchanged.


## Shared components

- Global lightbox behaviour lives in `docs/assets/javascripts/retro-home.js`.
- Shared component and navigation styling lives in `docs/assets/stylesheets/retro-home.css`.
- Image filenames have been normalised to be descriptive so future updates are easier.
- Reusable footer navigation now uses `retro-section-nav` and `retro-section-nav__link`.

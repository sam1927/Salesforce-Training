        <section class="retro-panel retro-step-panel is-active" data-step-panel>
          <h2>Overview</h2>
          <p>
            If you are going to be away, you can record this in Salesforce so it is visible to the team.
          </p>
          <p>
            If you need sessions to be <strong>cancelled</strong>, you will need to cancel them <strong>manually</strong>.
          </p>
          <p>
            This absence record is mainly there so <strong>Senior Leadership</strong> can see that you are away.
          </p>
          <p>
            For example, if a <strong>parent</strong> or <strong>carer</strong> gets in touch while you are on <strong>holiday</strong> or <strong>unwell</strong>, the team can check this first and avoid contacting you unnecessarily.
          </p>
        </section>

        <section class="retro-panel retro-step-panel" data-step-panel>
          <h2>How to Add an Absence</h2>
          <p>
            To record an <a href="https://mindjam.my.site.com/Portal/s/absences" target="_blank" rel="noopener">absence</a>:
          </p>
          <ul>
            <li>go to <strong>Appointment Management</strong></li>
            <li>open <strong>Absences</strong></li>
            <li>click <strong>New</strong></li>
            <li>enter the details</li>
            <li>click <strong>Save</strong></li>
          </ul>
          <p>
            Your absence will then be added to the system.
          </p>

          <div class="retro-image-panel">

        <section class="retro-panel retro-step-panel" data-step-panel>
          <h2>What This Is Used For</h2>
          <p>
            This does not automatically cancel sessions for you.
          </p>
          <p>
            Instead, it is mainly used so the <strong>Senior Leadership</strong> team can see that you are unavailable.
          </p>
          <p>
            This helps avoid confusion if someone tries to contact you while you are away.
          </p>
        </section>
